# 말씀의 발자취 v4

## v4 핵심 기능
- 닉네임당 하루 1회 인증
- 읽은 장 수와 관계없이 인증 1회 = 퍼즐 1조각
- 현재 퍼즐 닉네임별 기여 순위
- 관리자가 미래 퍼즐 사진·제목·조각 수·기간을 미리 등록
- 원하는 시점에 대기 퍼즐을 진행 중으로 전환
- 진행 도중 총 조각 수 수정 및 즉시 반영
- 현재 획득 조각보다 작은 값 입력 시 관리자 경고
- 완료 퍼즐 보관함
- 공개된 조각 클릭 시 인증 정보 확인

## 업로드 전 중요
1. `firestore.rules` 내용을 Firebase 콘솔 규칙에 게시하세요.
2. 퍼즐 사진은 GitHub 저장소의 `puzzles` 폴더에 먼저 업로드하세요.
3. 이 폴더의 전체 파일을 GitHub Pages 저장소에 덮어씌우세요.
4. 브라우저에서 Ctrl+F5로 새로고침하세요.

## 기존 기록 안내
v3 기록에는 `puzzleId`가 없으므로 v4의 현재 퍼즐 조각으로 자동 포함되지 않습니다. v4 적용 후 생성되는 기록부터 새 퍼즐 시스템에 연결됩니다.
