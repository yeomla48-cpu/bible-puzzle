# 말씀의 발자취 — 전체 최신본 v2

## 포함 기능

- GitHub Pages 배포
- Firebase 익명 로그인
- 관리자 이메일/비밀번호 로그인
- Firestore 성경 읽기 인증 저장
- 한 줄 묵상 저장
- 1장당 퍼즐 1조각 진행
- 관리자 퍼즐 사진 선택 UI
- 선택한 사진 미리보기
- GitHub `puzzles` 폴더 파일 자동 경로 생성
- 등록된 사진 선택 및 삭제
- Firestore 보안 규칙

## GitHub 업로드 방법

저장소 최상위에 아래 파일과 폴더를 모두 업로드해 기존 파일을 덮어씁니다.

- index.html
- style.css
- app.js
- firebase-config.js
- firestore.rules
- README.md
- puzzles/

## 퍼즐 사진 등록 방법

1. GitHub 저장소의 `puzzles` 폴더에 사진을 업로드합니다.
2. GitHub Pages 배포가 완료될 때까지 기다립니다.
3. 사이트 관리자 화면에서 **같은 원본 사진 파일**을 선택합니다.
4. 미리보기와 자동 경로를 확인합니다.
5. `퍼즐 사진 등록하기`를 누릅니다.

GitHub 파일명과 선택한 컴퓨터 파일명이 대소문자·띄어쓰기·확장자까지 같아야 합니다.

## 관리자

관리자 이메일은 `firebase-config.js`와 `firestore.rules`에 동일하게 설정되어 있습니다.
비밀번호는 Firebase Authentication에서 설정한 비밀번호를 사용합니다.


## v3 퍼즐 공개 방식

- 원본 사진 전체는 처음부터 보이지 않습니다.
- 읽은 장 수만큼 퍼즐 조각만 사진으로 공개됩니다.
- 아직 열리지 않은 조각은 빈 배경으로 표시됩니다.
- 1,189장을 모두 읽으면 전체 사진이 완성됩니다.
- 열린 조각은 클릭하면 해당 성경 읽기 기록을 확인할 수 있습니다.
